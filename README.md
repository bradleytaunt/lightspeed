# Light Speed

An insanely fast and performance-based Jekyll theme. Some fun facts about the theme:

* Perfect score on Google's Lighthouse audit
* Only ~434 bytes of CSS
* No media queries
* No JavaScript
* Probably the fastest Jekyll theme on the Internet
